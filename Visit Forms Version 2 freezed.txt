Visit forms:
   1. Visit forms can be created outside the beat
   2. Visit forms can be created within beat visit
   3. Visit forms can be created for the following types
            a. Existing Primary customer
            b. Existing secondary customer
            c. New customer
                       i. Primary customer
                      ii. Sub stockiest
                              1. Existing Market Expansion
                     iii. Outlet
                              1. New Market Expansion
                              2. Existing Market Expansion
   4. Once the day is started, user can view the “New visit form” button
   5. Once clicked on “New visit form” user can select the respective visit form and proceed
   6. Based on the selected visit form fields would appear
   7. In case if the visit form needs to be created outside the beat, visit form selection would be
       mandatory for the user and based on the visit form selected respective fields needs to be filled
   8. In case of “visit form” creation during the customer visit, customer name will be auto filled
       and remaining fields can be entered by the user
   9. Within the customer visit for Primary customer: TSE and above hierarchy users can create the
       visit form a. Existing primary customer and save
   10. Within the customer visit for secondary customer: DSM and above users can only create the
       visit form b. Existing secondary customer
   11. C. New customer visit forms can be created outside the beat only. Using the button “Create
       New visit form”
   12. TSE and above hierarchy users can create the visit forms for the customers mapped to them
       (Can search and select the customer for which they are mapped, statement applicable for
       existing customer visit form)
   13. DSM and above hierarchy users can create the visit forms for the customers mapped to them
       (Can search and select the customer for which they are mapped, statement applicable for
       existing customer visit form)
   14. If the visit form is created outside the beat using “New visit form” button system will capture
       the coordinates.
   15. KMS will be calculated and added to the worked beats (visits). This distance will be added to
       the total kms and it will shown as kms travelled under the expense management as well
   16. Each visit form with set of fields is shown below on the respective tables.
